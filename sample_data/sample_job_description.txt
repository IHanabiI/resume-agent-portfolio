岗位名称：AI 应用开发实习生

岗位职责：
1. 参与 AI Agent 应用原型开发，完成需求分析、功能设计和工程实现。
2. 使用 Python 开发数据处理、Web 应用和模型调用相关功能。
3. 基于大模型 API 构建智能问答、文档分析或自动化工作流。
4. 与产品、设计和业务同学协作，快速迭代可演示 Demo。

任职要求：
1. 熟悉 Python，有 Web 应用或脚本开发经验。
2. 了解 OpenAI API、LangChain、LangGraph 或其他 Agent 开发框架者优先。
3. 熟悉 SQL、数据清洗、接口调用等基础能力。
4. 有 Streamlit、Flask、FastAPI 等项目经验优先。
5. 具备清晰的文档意识和良好的沟通协作能力。
