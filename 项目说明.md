项目名称：简历定制 Agent

项目简介：
这是一个基于 LangGraph 和大模型的简历定制智能体。用户输入原始简历和目标岗位 JD 后，系统会自动分析岗位要求、提取简历经历、识别匹配点和缺失信息，并在生成简历前主动向用户追问必要上下文。最终输出一份不虚构、可追溯来源、适合目标岗位投递的定制化简历。

核心亮点：
1. LangGraph Agent 工作流。
2. Human-in-the-loop 追问机制。
3. 防虚构事实校验机制。
4. JD 关键词匹配与缺口诊断。
5. Markdown / DOCX 导出。
6. Streamlit 可视化界面。

推荐演示流程：
1. 运行 streamlit run app.py。
2. 使用 sample_data/sample_resume.txt 作为原始简历。
3. 使用 sample_data/sample_job_description.txt 作为目标岗位 JD。
4. 点击“开始分析”，查看岗位分析、候选人解析、匹配缺口和追问问题。
5. 根据追问补充真实信息，或填写“没有 / 跳过”。
6. 点击“生成定制简历”，查看定制简历、优化说明和事实来源映射表。
7. 下载 tailored_resume.md 和 tailored_resume.docx。
